# 4.2 Marketing - Ads Campaign - Ad Templates


1. Ad Name: The title of the campaign will be shown to WiFi users.
2. Description: The description of the campaign is displayed to WiFi users.
3. Image: The campaign image is shown to WiFi users. (JPEG, PNG, PDF, GIF, and many more)
4. Action Title: Set the desired action title, such as ‘Show More’ (This can be customized to fit your preferences).

To create Content templates, specify the Content Template Name, such as "Adopt. Rescue. Repeat." This name will be used as the campaign content and will be shown to connected WiFi users. There are several Template formats to select from the available options:

1. Image with Text
2. Video
3. Video with Text

Provide the necessary details for the content template:

- Title: Enter the title of the campaign that will be displayed to WiFi users.
- Description: The description of the campaign that will be shown to WiFi users.
- Image: Upload an image that represents the campaign. Accepted formats include JPEG, PNG, PDF, GIF, and more. (Please refer to recommended size)
- Action Title: Set the desired action title, such as "Show More." You have the flexibility to customize these settings

![](../../_images/42-marketing-ads-campaign-ad-templates-467.png)

As shown in the image above, utilize the following features:

1. **Language**:

The content can be set in various languages, including English, Traditional Chinese, and Simplified Chinese. This enables catering to a diverse audience and facilitating effective communication.

2. **Description Toolbar**:

The description toolbar offers a range of formatting options that enhance the appearance and emphasis of your content. These options include:

2.1) Bold/Italic/Underline

2.2) Text font & Size

2.3) Font Color & Highlight

2.4) Indention & Alignment

2.5) Bullet points & Numbering

2.6) Images

2.7) Emojis
