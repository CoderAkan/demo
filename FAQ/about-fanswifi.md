## Q: How is my customers’ private data protected and secured?

**A:** We are dedicated to ensuring the highest level of security by implementing a highly effective cybersecurity system. This includes a secure login system with HTTPS encryption for login credentials, a secure access login process that monitors user activity to detect suspicious behavior, mandatory two-factor authentication for all users, and a firewall to prevent unauthorized access and protect against cyber-attacks.

## Q: How can I restrict access to the platform's data dashboard to only authorized personnel?

**A:** To restrict access to the platform's data dashboard to only authorized personnel, we have implemented a secure login system. This includes requiring authorized users to enter a one-time password (OTP) to verify their identity and access permissions. Additionally, we have established role-based access controls, which allow users to view only the specific data relevant to their job responsibilities. This approach helps protect user privacy and prevent unauthorized access to sensitive information.
